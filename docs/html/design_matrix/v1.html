Design matrix

Type: object

A design matrix stored in a HDF5 file. The matrix itself is held in the design float dataset, where rows represent samples and columns represent model coefficients. Coefficient names are stored in the coefficients string dataset while sample names, if any exist, are stored in the samples string dataset. If the matrix was constructed from any factors, the names of the factors are saved in the factors string dataset.
