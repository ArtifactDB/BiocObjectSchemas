Data Frame


Virtual data frame object stored in a yet-to-be-defined file format. Columns with simple types are stored directly in the file. For columns with non-obvious types (e.g., nested data frames), their contents should be stored in other files, and a pointer to a resource should be stored in the corresponding entry of columns (a placeholder column may be created in the file, depending on the format).

root allOf item 0 data_frame columns columns items type
Type: enum (of string)

Type of the column.

  • Integers, (floating-point) numbers and booleans are their usual selves.
  • Strings have an optional format property that restrict their contents, e.g., for dates or times.
    This is only available in minor version 2 or higher. - The factor type is represented as an integer, to be used as a 1-based index into a vector of string levels. This type has an additional levels property specifying the levels, as well as an ordered property indicating whether they are ordered.
    • Older instances (minor version 1) of data_frame representations store factor and ordered types as strings instead of integers. All such strings are guaranteed to belong to the string levels in levels. This representation is deprecated and the integer representation should be used in minor version 2 and higher.
      • The ordered type is a deprecated alias for the factor type with the ordered property set to true; the latter should be used in minor version 2 or higher
      • The date type is a soft-deprecated alias for the string type with format property set to date; the latter should be used in minor version 2 or higher
      • The date-time type is a soft-deprecated alias for the string type with format property set to date-time; the latter should be used in minor version 2 or higher
  • Columns listed as other are assumed to be non-simple and should contain a resource property pointing to column's contents.

Must be one of:

  • "integer"
  • "number"
  • "string"
  • "factor"
  • "ordered"
  • "boolean"
  • "date"
  • "date-time"
  • "other"
