Simple list

Type: object

Virtual class representing a simple list of objects. The exact representation of the list on disk is left to concrete subclasses, as long as it supports external references to child resources. These external references should be used for complex objects that are not accommodated by the list representation itself. Each external reference should be an index into the simple_list.children array.
